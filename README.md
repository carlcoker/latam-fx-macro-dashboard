# latam-fx-macro-dashboard

A browser based macro dashboard for monitoring selected LatAm FX pairs against key external drivers including DXY, US Treasury yields, oil, and copper.

## Coverage

- MXN
- BRL
- COP
- CLP
- PEN

## Features

- live FX and macro panels
- macro score framework
- correlation matrix
- mispricing heatmap
- system signal panel
- single file browser implementation

## Notes

This is a prototype dashboard designed to visualise macro factor relationships and relative value conditions in LatAm FX.
